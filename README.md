# Drag Buddy

A lightweight macOS menu bar utility that lets you drag any window from anywhere by holding a modifier key.

[![Download on Gumroad](https://img.shields.io/badge/Download-Gumroad-ff90e8?style=for-the-badge&logo=gumroad&logoColor=white)](https://felipemendes.gumroad.com/l/drag-buddy)

## Features

- **Drag windows from anywhere** - Hold your chosen modifier keys and drag any part of a window to move it
- **Customizable key combinations** - Use Option+Control, Option+Command, Control+Command, or set your own custom combination
- **Menu bar app** - Lives quietly in your menu bar, always ready when you need it
- **Activate window on drag** - Optionally bring windows to the front when you start dragging
- **Launch at login** - Start automatically when you log in
- **Lightweight** - Minimal resource usage, no background processes

## System Requirements

- macOS 11.5 (Big Sur) or later
- Accessibility permission required

## Installation

1. Download Drag Buddy from [Gumroad](https://felipemendes.gumroad.com/l/drag-buddy)
2. Move the app to your Applications folder
3. Launch Drag Buddy
4. Grant Accessibility permission when prompted (required for window dragging to work)

## Usage

1. Click the Drag Buddy icon in your menu bar to access settings
2. Hold your configured modifier keys (default: Option + Control)
3. Click and drag anywhere on a window to move it

## Preferences

Access preferences from the menu bar icon or press `Cmd + ,`:

- **Display icon** - Choose where to show the app icon (Menu Bar, Dock, or both)
- **Key combination** - Set your preferred modifier keys for dragging
- **Enable dragging** - Toggle the drag feature on/off
- **Activate window on drag** - Bring windows to front when dragging
- **Launch at login** - Start Drag Buddy automatically

## Privacy & Security

Drag Buddy requires Accessibility permission to detect modifier keys and move windows. The app:

- Never logs keystrokes
- Never inspects window contents
- Never captures screenshots
- Never accesses the network
- Never collects any data
